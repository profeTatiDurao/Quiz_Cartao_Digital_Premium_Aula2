Quiz - Conteúdo e Ícones do Cartão Digital

Responda às perguntas sobre links, ícones, grid, hover e rodapé do cartão digital.

1) No trecho abaixo, qual é a função do atributo href?

<a class="icone" href="tel:+5514999999999">




2) Observe este link:

<a class="icone" href="https://wa.me/5514999999999" target="_blank">

Qual alternativa interpreta melhor o uso de target="_blank"?




3) No CSS, a área dos ícones foi organizada assim:

.icones { display: grid; grid-template-columns: 1fr 1fr; gap: 14px; }

Qual é o efeito visual mais provável desse código?




4) Analise estes dois trechos:

.icone:hover { background: #172033; color: white; }

.icone:hover i { color: white; }

O que acontece quando o usuário passa o mouse sobre um ícone?




5) Analise o CSS do rodapé:

.rodape { background: #0d1528; color: white; text-align: center; padding: 18px; font-size: 13px; }

Qual alternativa explica melhor o papel visual do rodapé no cartão digital?
